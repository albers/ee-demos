scopeView.xhtml
Die Anwendung verwendet einige verschieden 'gescopte' Objekte, die jeweils einen Zähler enthalten, der anzeigt, das wievielte Objekt seiner Art seit Anwendungsstart es jeweils ist. Die Objekte werden von zwei Beans ()scopeModel und someOtherBean) referenziert.
Mit den Buttons kann ein Reload der Seite veranlasst werden, wobei die beiden rechten eine Konversation beginnen bzw. beenden. Beobachte Sie, wie sich dabei die Objekte ändern.
Eine neue Session können Sie durch Schliessen und Neuladen des Browsers erzwingen.
